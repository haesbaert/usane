module Usane: sig .. end
Sane unsigned integers.

Instead of automatically wrapping and leaving it up to the caller to detect wrapping around, this library provides the information whether the operation lead to a overflow explicitly. Each operation results in a tuple of the concrete value t and a bool whether it overflew. The implementation uses C compiler builtins (__builtin_uadd_overflow etc.) if available (since GCC 5.1 and Clang 3.4).

It is not a replacement for arithmetic operations in OCaml, but designed for network protocols where often sequence numbers are used (and either wrapping around is accepted or should lead to re-establishing or re-keying of the connection).
